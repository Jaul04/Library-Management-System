📚 Library Management System

A modern and responsive Library Management System built using the MERN Stack technologies (Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript). This project helps libraries manage books, students, user authentication, book issuing, and return records through a simple and user-friendly interface.

🚀 Features
🔐 User Signup & Login Authentication
📚 Book Management (Add, Update, Delete, View Books)
👨‍🎓 Student Management
📖 Issue & Return Books
🔍 Search Books and Students
📱 Fully Responsive User Interface
🎨 Modern UI with HTML, CSS & JavaScript
🗄️ MongoDB Database Integration
🌐 Deployed on Render
